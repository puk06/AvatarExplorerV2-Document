# VRC Avatar Explorer V2

VRC Avatar Explorer V2 は、VRChatユーザー向けのクロスプラットフォーム対応アセット管理ツールです。

Boothのアイテム情報とアセットファイルをひも付けることで、エクスプローラー形式で分かりやすく整理できます。

:::tip こんな人におすすめ
- 「購入したアセットが増えて管理が大変」
- 「どのアバターに対応しているかを忘れがち」
- 「Unitypackageの導入作業をもっと効率化したい」
:::

## 主な機能

- アセットファイルの自動整理: アイテム情報に基づいて見やすく分類
- 圧縮ファイルの自動展開: `zip`/`rar`/`7z`/`gz`/`tar` に対応
- 高速検索: タイトル、作者、カテゴリなどからすぐに検索
- 対応アバター管理: アセットごとの対応先を一覧管理
- 共通素体グループ管理: 共通素体単位で扱いやすく整理
- 独自のUnitypackageインポート: パス調整や複数ファイル一括導入に対応
- ステータス管理: タグ、メモ、実装状況を細かく管理
- 背景カスタマイズ: 好みの画像を背景に設定可能
- アプリから外部へのD&D: Unityなどへ直接ドラッグ&ドロップ
- 仮登録アバター: 未登録アバターを一時的に追加して運用可能

## はじめる前に

まずはこの順番で進めるのがおすすめです。

1. ダウンロードして起動
2. アイテムを追加
3. 設定画面を開いて初期設定
4. 必要に応じて共通素体や一括インポートを活用
