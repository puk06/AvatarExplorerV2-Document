# 一括インポート機能

このページでは、VRC Avatar Explorer V2 の一括インポート機能を説明します。

## 一括インポート機能とは

複数のUnitypackageをまとめて管理し、効率よくインポート作業を進められる機能です。

## 使い方

一括インポート機能は、画面右サイドパネルの上から2番目にあります。

1. 追加したいアイテムを登録する
2. 必要に応じて並びや構成を調整する
3. インポートを実行する

追加方法は次の2つです。

- アイテムを右クリックし `一括インポートに追加` を選択
- アイテムをドラッグして一括インポート画面へドロップ

## 一括インポートプリセット機能について

一括インポートに追加した構成は、プリセットとして保存できます。

プリセット画面は、右サイドパネルの上から3番目のアイコンから開けます。

保存は、アイテム一括インポート画面の `保存` ボタンから行います。

保存したプリセットはデータベースに記録されるため、アプリを閉じても保持されます。

削除する場合は、対象プリセットを右クリックして `削除` を選択してください。

:::tip プリセット活用例
アバター別や作業内容別にプリセットを分けておくと、導入作業の再現性が上がります。
:::

:::warning 注意
大量のパッケージを同時に扱うと処理負荷が高くなるため、ほかの重い作業と同時実行しないのがおすすめです。
:::
