# 他ツールからのインポート

このページでは、他ツールのデータを VRC Avatar Explorer V2 に取り込む方法を説明します。

## 対応ツール

- VRC Avatar Explorer V1
- KonoAsset

## インポート方法

1. 画面左上の設定アイコンをクリックして、設定画面を開きます。
2. `データ` タブを選択します。
3. `他ツールからのインポート` の右にある `インポート` ボタンをクリックします。
4. インポートしたいツールを選択します。
5. インポートしたいデータのフォルダを選択します。

フォルダ選択の目安は次のとおりです。

- VRC Avatar Explorer V1: V1のデータフォルダ
- KonoAsset: 保存先として指定しているフォルダ（`metadata` などが入っているフォルダ）

処理が完了すると、アイテムが追加されます。

:::warning 処理負荷について
インポートはディスク容量とCPUを大きく使います。重要な作業中は避け、余裕のあるタイミングで実行してください。
:::

:::tip 失敗しやすいポイント
取り込み元のフォルダ階層がずれていると、データを正しく検出できないことがあります。選択前にフォルダ構成を確認してください。
:::
