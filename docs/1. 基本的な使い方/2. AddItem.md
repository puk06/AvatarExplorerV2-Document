# アイテムの追加

このページでは、VRC Avatar Explorer V2 にアイテムを追加する方法を説明します。

## 追加方法は2つ

- 追加したいファイル/フォルダをアプリへドラッグ&ドロップ
- 画面右上の「アイテムを追加する」から追加

:::tip おすすめ
ファイルとフォルダを混在でまとめて入れられるため、ドラッグ&ドロップのほうが効率的です。
:::

## 基本の登録フロー

1. ファイルまたはフォルダを追加する
2. Booth URLを `Booth URL` 欄に入力し、`情報取得` を押す
3. カテゴリを選択する（必要ならカスタムカテゴリを追加）
4. 対応アバターを設定する

## 追加時のフォルダ利用について

標準ではアセットファイルは追加時にコピーされますが、`設定 > アイテム > フォルダをそのまま使用する` をオンにすると、渡されたフォルダの1つ目がフォルダだった場合はコピーせずにそのフォルダをそのまま使用します。

## 対応アバター設定の補足

- 既存アバターを選ぶ: `対応アバター` ボタンから選択
- 未登録アバターを使う: 左下の `仮登録アバターの追加` から作成して選択

:::info 後から編集できます
登録した情報は、アイテムを右クリックして `アイテムを編集する` からいつでも更新できます。
:::

:::warning 情報取得がうまくいかないとき
Booth URLの形式が正しいか確認してから、再度 `情報取得` を実行してください。
:::
