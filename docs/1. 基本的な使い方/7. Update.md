# アップデート

このページでは、VRC Avatar Explorer V2 を最新バージョンへ更新する手順を説明します。

## 手順

1. [最新のリリースページ](https://github.com/puk06/VRC-Avatar-Explorer/releases/latest)を開きます。
2. 使用しているOS向けのファイルをダウンロードします。
3. Windowsの場合は、`setup.exe` を実行して更新します。
4. Windows以外の場合は、解凍後にフォルダ内の `AvatarExplorer` を実行します。

## 更新後の確認

- アプリが正常に起動する
- 画面表示に崩れがない
- これまでの操作が問題なく行える

:::warning 更新前の注意
更新前にアプリを起動している場合は、いったん終了してから実行してください。
:::

:::tip うまく更新できない場合
ダウンロードファイルが壊れている可能性があります。ファイルを再取得して、もう一度実行してください。
:::
